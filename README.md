# messenger-shortcuts

> Chrome extension which adds keyboard shortcuts to messenger.com

Some awful JS I wrote a long time ago which I still use every day.

Adds the following keyboard shortcuts to messenger.com:

 ### Next chat
 
  Windows: `TAB`
  
  Mac: `TAB`
  
### Previous chat

Windows: `SHIFT + TAB`

Mac: `SHIFT + TAB`
 
### Focus search box

Windows: `ALT + S`

Mac: `CTRL + F`

### Focus message entry

Windows: `ALT + L`

Mac: `CTRL + L`
